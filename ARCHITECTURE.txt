┌──────────────────────────────────────────────────────────────────────────────┐
│                          AUTHENTICATION FLOW                                 │
└──────────────────────────────────────────────────────────────────────────────┘

Request Received
    │
    ├─ Check session for user_id
    │
    ├─ If NOT logged in:
    │  └─ Redirect to /customers/ (public page)
    │     └─ Display base_public.html (with Login button)
    │
    └─ If logged in:
       ├─ Check user role in session
       │
       ├─ If role == "owner":
       │  └─ Redirect to /owners/dashboard/
       │     └─ Display base_owner.html (with owner sidebar)
       │
       ├─ If role == "tenant":
       │  └─ Redirect to /tenants/dashboard/
       │     └─ Display base_tenant.html (with tenant sidebar)
       │
       └─ If role == "customer":
          └─ Redirect to /customers/
             └─ Display base_public.html (with logout option)
